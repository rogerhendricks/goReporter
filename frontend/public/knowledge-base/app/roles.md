# Roles (Admin / Doctor / User)

This knowledge base is backed by Markdown files in `frontend/public/knowledge-base/`.

## Admin

Admins typically manage configuration and operations:

- Manage doctors, patients, devices, leads
- Manage tags and task templates
- Run productivity and reporting workflows

## Doctor

Doctors typically review and act on clinical information:

- View assigned patients
- Review reports and device data
- Use tasks to track follow-ups

## User

Users typically help with day-to-day operations:

- Create and update patients
- Create and complete tasks
- Build reports (including tag-based filters)

## Tips

- Use the global search (top bar) to quickly jump to patients.
- Use Tasks to track work and follow-ups.
