# Task Team Assignment

## Overview
Assign tasks to individual users or entire teams for better workload distribution and collaboration.

## Who Can Use This

**Administrators & Doctors:**
- Create tasks assigned to users or teams
- Reassign tasks between users and teams
- View all tasks in the system

**Regular Users:**
- View tasks assigned to them
- View tasks assigned to their team(s)
- Cannot reassign tasks

## Assignment Options

### Individual Assignment
- Task assigned to specific person
- Clear individual accountability
- Best for personal follow-ups

### Team Assignment
- Task assigned to entire team
- Any team member can work on it
- Shared responsibility
- Best for collaborative work

## How to Assign Tasks

### Creating a New Task

1. **Click "New Task"**
2. **Choose assignment type:**
   - **User Tab:** Select individual from dropdown
   - **Team Tab:** Select team from dropdown
3. **Fill in task details**
4. **Set due date and priority**
5. **Submit**

### Reassigning Existing Tasks

**Admins/Doctors can:**
1. Open task details
2. Click "Edit"
3. Switch between User/Team assignment
4. Select new assignee
5. Save changes

## What Team Members Can Do

When a task is assigned to your team:
- **View** the task in your task list
- **Update** task status and progress
- **Add notes** and comments
- **Mark complete** when finished

**Any team member** can take action - no need to claim or assign to yourself first.

## Viewing Tasks

### Task List Shows:
- **Individual assignments:** User's name
- **Team assignments:** Team name with color badge
- **Created by:** Who created the task

### Filtering
- Filter by assigned user
- Filter by assigned team
- See only your tasks or all tasks (based on role)

## Best Practices

### When to Assign to a Team
- Tasks requiring multiple skill sets
- Department-wide responsibilities
- On-call rotations
- Work that any qualified person can handle

### When to Assign to an Individual
- Tasks requiring specific expertise
- Personal follow-ups
- Clear individual accountability needed
- Time-sensitive for specific person

### Team Management Tips
- Keep teams small and focused (3-10 members)
- Assign teams based on expertise areas
- Use team colors for easy visual identification
- Regularly review team workload

## Benefits

**For Managers:**
- Better workload distribution
- Visibility into team assignments
- Flexibility for complex tasks

**For Team Members:**
- Shared responsibility
- Any member can respond
- Better collaboration

**For Workflow:**
- Multi-person tasks handled smoothly
- Department tasks routed appropriately
- No single points of failure

## Troubleshooting

**Can't see a team-assigned task?**
- Verify you're a member of that team
- Check with admin if team membership is correct

**Can't assign to a team?**
- Only admins and doctors can assign to teams
- Regular users can only create self-assigned tasks

**Task shows wrong assignment?**
- Contact an admin to reassign
- Check that only one assignment type is set (user OR team)
