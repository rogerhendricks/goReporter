# Report Tags

## Overview
Organize and categorize device reports using custom tags. Tags help you quickly find related reports and track different types of findings.

## Who Can Use This
- **All users** can view and apply tags to reports
- **Administrators** can create and manage tag definitions

## Tag Types

### Patient Tags
- Applied to patient records
- Examples: "High Risk", "Post-Op", "Research Study"
- Used across all patient interactions

### Report Tags
- Applied to individual device reports
- Examples: "Abnormal Findings", "Battery ERI", "Programming Changed"
- Help track report-specific findings

## Using Tags

### Applying Tags to Reports

1. **Open a report** (create new or edit existing)
2. **Scroll to Tags section**
3. **Select relevant tags** from dropdown
4. **Save the report**

### Viewing Tagged Reports

- Tags appear on report detail pages
- Filter reports by tag in search
- Color-coded for quick visual identification

### Creating New Tags (Admin Only)

1. **Go to Admin Dashboard â†’ Tag Management**
2. **Click "Create Tag"**
3. **Enter tag name**
4. **Select type:** Patient Tag or Report Tag
5. **Choose color** for visual identification
6. **Add description** (optional)
7. **Click Create**

## Common Report Tag Examples

| Tag | Use Case |
|-----|----------|
| **Abnormal Findings** | Reports with unusual device behavior |
| **Battery ERI** | Battery at Elective Replacement Indicator |
| **Battery EOL** | Battery at End of Life |
| **Programming Changed** | Device settings were modified |
| **Lead Issue** | Lead-related concerns identified |
| **Arrhythmia Events** | Significant rhythm events recorded |
| **Follow-up Required** | Needs additional attention |
| **Routine Check** | Standard follow-up with no issues |

## Benefits

**For Doctors:**
- Quickly identify reports needing attention
- Track patterns across multiple reports
- Filter reports by specific findings

**For Staff:**
- Standardize report categorization
- Ensure consistent follow-up protocols
- Generate reports by tag category

**For Administrators:**
- Monitor tag usage statistics
- Identify common issues
- Quality assurance tracking

## Tips

- Use consistent tagging practices
- Create tags for your clinic's specific needs
- Color-code by severity (red for urgent, green for normal)
- Train staff on when to use each tag
